# 📊 Calculateur de Moyenne - L3 S1/S2

Ce projet est une application web permettant de calculer la moyenne des étudiants de L3 (S1 et S2) en fonction des modules, des contrôles et des travaux continus.

---

## 🚀 Fonctionnalités

- Calcul automatique de la moyenne par module
- Prise en compte des coefficients
- Gestion des pourcentages (CC / Contrôle)
- Calcul de la moyenne générale (S1 et S2)
- Interface simple et facile à utiliser
- Améliorations progressives de la structure du projet

---

## 🛠️ Technologies utilisées

- HTML5
- CSS3
- JavaScript
- PHP (si utilisé dans certaines versions)
- XAMPP (environnement local)
